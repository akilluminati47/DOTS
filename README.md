# DOTS ● Beta Releases

  This repo is the public home for DOTS beta builds.
  
  Current releases are now published as GitHub Release assets and served to
  all platforms via the website download links below.

  **→ [Play online + view the leaderboard](https://dots-online.pages.dev)**

  Pop dots. 300 levels. One clock.

  DOTS is a time trial puzzle game — clear every level as fast as possible, then check the global leaderboard to see
  where your run lands. The leaderboard tracks your full prestige run time, updated live across all platforms.

  ## About

  - 300 levels across a single prestige run
  - Time tracked from level 1 through level 300
  - Colors, glyphs, and tags unlock as you progress
  - Top 3 finishers marked with α β γ on the board

  Report bugs or feedback via GitHub Issues.
